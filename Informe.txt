Este programa en java permite registrar varios productos tecnológicos y calcular los impuestos aplicados a cada uno.
Primero el sistema pide cuantos productos se van a ingresar y el dia de operacion, validando que un dia entre 1 y 7. 

Luego por cada producto, se solicita el nombre, precio base, categoria, si tiene riesgo tecnológico y la cantidad en stock
Segun la categoria seleccionada el programa no aplica ciertos impuestos.Por ejemplo, algunos productos sin IVA paganom los que tiene riesgo tecnologico generan impuesto IRT,
y si el registro se hace sabado o domingo tambien se aplica el TCE. Cada impuesto se calcula tomando como base el precio ingresado, con esos valores se forma el precio final
del producto.

Mientras se ingresan los productos, el programa va acumulando el total de cada impuesto, el monto total del dia, cuantos productos pertenecen a cada categoria y cual fue el 
producto mas caro. Al final muestra un reporte con todo lo recaudado, la categoria con mas registros y el producto que obtuvo el precio mas alto.

En resumen esto es un sistema sencillo de registro que organiza los datos, calcula el impuesto según las condiciones especificas y genera un informe claro y completo
